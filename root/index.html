About the FHIR Standard

The Fast Healthcare Interoperability Resources (FHIR) standard is a modern format for exchanging healthcare data electronically. It is designed to be easy to implement and use, while also supporting a wide range of use cases.

The HL7 Estonia FHIR Standard is a customized implementation of FHIR that is specific to the needs of the Estonian healthcare system. It includes profiles and extensions that are tailored to Estonian healthcare data and workflows.
