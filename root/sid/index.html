SID

https://fhir.ee/sid is root directory for system identifiers.

System identifiers have the hierarchical structure:

\pid - person or patient identifiers

\pid\est - person or patient identifiers issued in the Estonia.

\pid\est\ni - Estonian Personal National Identification Code (isikukood).

\org - organization identifiers
